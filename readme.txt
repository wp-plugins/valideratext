Valideratext.se för WordPress via Flowcom AB

== Description ==
This plugin uses the Swedish service Valideratext.se. Therefore there is no other language support in this description or in the plugin.
- - -

Koppling till valideratext.se (API v3) för redaktionellt stöd. Valideratext.se hjälper dig att skriva texter som är enkla och lätta att förstå.

Du som redaktör och textförfattare i WordPress som verktyg kan behöva stöd i dina textutformningar på webben. Valideratext är en tjänst som du prenumererar på och den hjälper dig på ett smart sätt med det.
Skriv din text och klicka på knappen i WordPress så kommer texten att valideras och förslag på förbättringar att visas i ett eget fönster.
Du kan därifrån redigera din text och testa den igen mot Valideratext.

Om du har prenumeration kan du mata in dina uppgifter för tjänsten. Om inte kommer du att kunna testa den på en delmängd av texten för att se att det fungerar bra.

Ladda ner den nu och använd den i din WordPress-installation direkt.

Demo: Du kan testa denna plugin via webbsidan http://plugins.flowcom.se/valideratext, kontakta oss för inloggning!

Vid kontakt angående tekniken i plugin kontakta Andreas Ek på Twitter: @EkAndreas

Roadmap: Nästa version, administratörsinställningar för dig som prenumererar på tjänsten.


== Installation ==

Ladda ner till din WordPress-installation på vanligt vis via WordPress offentliga repository.
Aktivera denna modul via "tillägg".
Använd funktionen i din redigerare, det är en egen knapp för funktionen.

== Screenshots ==

1. En ikon läggs till i WordPress-standardredigerare. Redigera text och klicka för att validera din text!
2. Valideratext analyserar din text och ger dig instruktioner och statistik för att öka läsbarhet och förståelse.
